# LÖVE RPG Creator
### By Julio :)

## What is this?  

LÖVE RPG Creator (LRPGC) is a set of libraries which make creating RPG's much easier in the Lua's framework, LÖVE. These are free to use, licensed under the MIT license, and built in a modular way so you can use these anywhere.

## Whats included?

LRPGC includes many utilities to accelerate your workflow, such as:

- A basic timer
- RPG-style textbox with typewritter effect
- Inventory system
- Sample for main character and a party
- Sample for enemies
- Sample for RPG-style turn based combat
